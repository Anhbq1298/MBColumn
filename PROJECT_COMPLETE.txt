TECHNICAL DETAILS
================================================================================

Dual Tolerance Logic:
  P PASSES if: (|%DiffP| <= 5%) OR (|DiffP| <= 25 kN)
  M PASSES if: (|%DiffM| <= 5%) OR (|DiffM| <= 25 kNm)
  POINT PASSES if: P_PASS AND M_PASS

Near-Zero Handling:
  If |RefValue| < 1e-6:
    Skip percentage calculation
    Judge by absolute tolerance only

Point Matching:
  Priority 1: Index-based (fast, deterministic)
  Priority 2: Nearest-neighbor (robust, normalized P-M distance)
  Both clearly documented in report

Unit System:
  Internal: N, mm, N-mm, MPa
  Reference CSV: kN, kNm (auto-converted)
  Output PDF: kN, kNm

Statistics (per theta):
  Mean, Variance, Stddev: %DiffP, %DiffM
  Max Abs: DiffP, DiffM
  Pass/Fail counts

Performance:
  Test runtime: <2 seconds
  Memory: ~5-10 MB
  PDF size: ~1 MB
